# Preprocessing

## Requirements

This implementation has been tested with the following versions.

```
python 3.6.2
mecab-python 0.996
zenhan 0.5.2
tqdm 4.28.1
```

## How to Use
基本的な前処理プログラム  

Count.py  
単語数・語彙数等をカウントするプログラム．  

Delete.py  
文字列削除に関するプログラム．  

File_operation.py
ファイル操作に関するプログラム．  

Sentence.py  
文書を1文に分けるプログラム．  

Wakati.py  
形態素解析に関するプログラム．

Object  
対象毎の前処理プログラム．
